# Kaggle AI Agents Vibe Coding Capstone Project

Welcome to the Kaggle AI Agents Vibe Coding Capstone repository! This project implements an agentic workflow for the **Agents for Business** track.

## Project Structure
* **`docs/`**: Project design proposals and architecture diagrams.
  * **[capstone_project_proposals.md](docs/capstone_project_proposals.md)**: Detailed breakdown of the two proposed architectures:
    1. **Smart Airline Turnaround & Delay Mitigation Agent** (using the US DOT Flight Delays dataset)
    2. **E-Commerce Logistics Fulfillment & SLA Breach Predictor** (using the Olist Brazilian E-Commerce dataset)

## Next Steps
1. Select the final proposal to build.
2. Initialize the SQLite database and seed the dataset.
3. Scaffold the multi-agent system using the Google Agent Development Kit (ADK).
